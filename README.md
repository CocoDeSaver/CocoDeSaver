# 🧠 Featured Project

## AI Medical Chatbot (RAG + IndoBERT + LLM)

An experimental AI chatbot designed to analyze user symptoms and generate medical information using a hybrid AI pipeline.


### Key Ideas

- Symptom classification using **IndoBERT**
- Knowledge retrieval using **RAG architecture**
- Response generation using **Large Language Models**
- AI system pipeline combining **classification + retrieval + generation**

This architecture helps improve response relevance and reduces hallucination compared to pure LLM-based chatbots.
